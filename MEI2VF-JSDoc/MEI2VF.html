<static> render_notation(xmlDoc, target, width, height, backend, options)

Basic rendering function. Uses the m2v.Converter's prototype as a singleton. No scaling; page layout information in the MEI code is ignored.
Parameters:
Name Type Description
xmlDoc XMLDocument The MEI XML Document
target XMLElement An svg or canvas element
width Number The width of the print space in pixels
height Number The height of the print space in pixels
backend Number Set to Vex.Flow.Renderer.Backends.RAPHAEL to render to a Raphael context; if falsy, Vex.Flow.Renderer.Backends.CANVAS is set
options Object The options passed to the converter. For a list, see MEI2VF.Converter
